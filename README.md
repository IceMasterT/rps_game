# Rock Papers Scissors game Implementation


## Version 1

### Features
- Multiple players can create an instance of game and play ✅ 
- can deposit bet amount if player ✅ 
- can submit move ✅ 
- Incentivizes the winner ✅ 
- Should reset the game if moves are same ✅
